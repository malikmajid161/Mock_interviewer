# Interview Forge Pages

This directory contains the React components for each view in the Interview Forge application.

- `Landing.jsx`: The public landing page.
- `SignUp.jsx`: User registration with progress tracking.
- `SignIn.jsx`: Secure login page.
- `Verify.jsx`: OTP email verification.
- `Profile.jsx`: Post-registration profile completion.
- `Dashboard.jsx`: Main user interface after login.
